## Conventions

- **UTCS layout** inside each CA: `UTCS-01-Requirements/` … `UTCS-11-Sustainment-Recycle/`.  
- **Artifacts-as-code**: keep YAML/JSON, analyses, and test logs under the appropriate UTCS folder.  
- **Cross-domain traceability**: reference CAs/CIs explicitly (e.g., `CI-CA-A-001-001`) in requirements, ICDs, and tests.
